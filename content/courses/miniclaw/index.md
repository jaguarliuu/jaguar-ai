MiniClaw 是一门面向工程实现的中文课程。它不依赖黑盒框架，而是从开发环境、基础设施、LLM 客户端到实时通信能力，逐层搭建一个可维护、可扩展的 Agent 系统。

当前站点先公开第 3 章到第 5 章，分别对应环境与基础底座、从零手写 LLM 客户端，以及 WebSocket 网关与实时通信。后续章节会继续通过 Git 方式增量更新。

## 课程结构

- [第3章：开发环境与基础底座](/courses/miniclaw/chapter-03)
- [第4章：从零手写 LLM 客户端](/courses/miniclaw/chapter-04)
- [第5章：WebSocket 网关与实时通信](/courses/miniclaw/chapter-05)

## 这门课适合谁

- 想系统理解 Agent 基础设施，而不是只会调用框架的人
- 希望把 LLM 集成、实时通信和工程组织方式串起来的人
- 正在搭建个人产品或课程内容，需要一套完整后端骨架的人

## 学习方式

建议按章节顺序学习。每个章节先看概览，再进入小节正文；这样能先建立整体结构，再理解每一步实现的取舍。
